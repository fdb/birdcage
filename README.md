Birdcage
========
Birdcage stores Twitters tweets.

Birdcage uses node.js and stores tweets in MongoDB (in the "birdcage" database).

To run, make sure you have installed node.js, npm and MongoDB, then run:

    cd birdcage
    npm install
    node app <search term>
